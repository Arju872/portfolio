This project demonstrates how to deploy a scalable web application on Amazon Elastic Kubernetes Service (EKS) using an Ingress Controller to manage external traffic efficiently. The goal is to automate infrastructure creation, container deployment, and expose the application securely to the internet.

EKS provides a managed Kubernetes environment, while the Ingress Controller (NGINX) helps handle HTTP routing and load balancing between services.
